# Odoo Debug Mode Toggle Extension

A Chrome extension that helps manage Odoo debug modes with ease.

## Features

- Toggle between three debug modes:
  - No debug
  - Normal debug (`debug=1`)
  - Debug assets (`debug=assets`)
- Automatic detection of current debug mode from URL
- One-click switching between modes
- Visual feedback with mode-specific icons
- Auto-closing popup after 2 seconds
